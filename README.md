# Projeto Contador React

## Descrição

Este projeto é uma aplicação simples desenvolvida em React que implementa um contador de cliques.

Cada vez que o botão é clicado, o valor do contador é incrementado em 1.

O objetivo deste projeto é praticar conceitos básicos do React, como:

- Componentes funcionais
- useState
- Eventos com onClick
- Renderização dinâmica

------------

## Tecnologias Utilizadas

- React
- JavaScript
- CSS
- Create React App

## Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

### 2. Acesse a pasta do projeto

```bash
cd meu-projeto
```

---

### 3. Instale as dependências

```bash
npm install
```

---

### 4. Execute o projeto

```bash
npm start
```

O projeto abrirá automaticamente no navegador em:

```bash
http://localhost:3000
```
